# Examen Programación Híbrida – Ionic + Angular

Aplicación híbrida desarrollada con **Ionic y Angular** como parte del examen de Programación Híbrida.

##  Funcionalidades
- Crear publicaciones con formulario validado
- Listar publicaciones almacenadas localmente
- Eliminar publicaciones con ventana de confirmación (modal)
- Persistencia de datos mediante **Capacitor Preferences**
- Uso de componentes Ionic y directivas de Angular
- Formateo de fecha mediante pipe

##  Tecnologías utilizadas
- Ionic
- Angular
- Capacitor Preferences
- TypeScript

##  Estructura
El proyecto se organiza en componentes, servicios y páginas para una correcta separación de responsabilidades.
